# Vivestom
Vivestom is a companion library for Vivecraft, the VR mod for Java Minecraft.

## Usage

```groovy
repositories {
    maven {
        name = "mirageRepository"
        url = "https://repo.mirage-play.com/releases"
    }
}

dependencies {
    implementation "me.playgamesgo:vivestom:1.0.0"
}
```
